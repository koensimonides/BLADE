# KernelTuner
* This task focuses on designing novel metaheuristic algorithms for solving kernel tuning optimization problems under realistic constraints. The optimization setting reflects challenges in natural computing and hardware-aware kernel configuration, including:
  * Integer and/or continuous parameters
  * Variable-dimensional search spaces
  * Parameter constraints
  * Black-box objective evaluation

* The goal is to implement a robust and general-purpose optimizer capable of efficiently exploring heterogeneous search spaces.
* Implements a class that inherits from a custom class in `OptAlg`.
